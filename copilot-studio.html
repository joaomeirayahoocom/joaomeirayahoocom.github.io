Copilot Studio Employee Self Service Agent

AI-powered virtual assistant for HR policy inquiries with SharePoint integration

Published: November 9, 2025 | Status: Production Ready

Project Overview

I developed an intelligent Employee Self Service Agent using Microsoft Copilot Studio to automate HR policy inquiries and reduce the workload on HR staff. The agent leverages generative AI to extract information from SharePoint-hosted policy documents and provides instant, accurate responses to employee questions about vacation policies, PTO balances, and carry-over rules.

The agent integrates seamlessly with existing SharePoint document repositories, eliminating the need for manual policy lookups and ensuring employees receive consistent, up-to-date information 24/7.

Business Challenge

HR departments face repetitive inquiries about vacation policies, PTO balances, and time-off procedures. These questions consume significant HR staff time that could be better spent on strategic initiatives. Employees often experience delays waiting for responses, and information inconsistency can lead to confusion and frustration.

Solution: Create an AI-powered self-service agent that provides instant, accurate answers by directly querying policy documents stored in SharePoint.

Technical Implementation

Topics Configured

I created 5 custom topics to handle specific user interactions:

  • Greeting: Welcomes users and introduces the agent's capabilities
  • Prompt Vacation Days: Core topic that handles vacation policy queries, balance lookups, and carry-over rules
  • Start Over: Allows users to reset the conversation and begin a new query
  • Thank You: Acknowledges user gratitude and offers additional assistance
  • Goodbye: Provides friendly closure to conversations

Additionally, 9 system topics handle standard conversation management, escalation, and error handling.

Knowledge Sources

The agent is connected to authenticated SharePoint sites containing HR policy documentation:

  • Documents Site: General HR policy repository
  • rmgenvm.cap426261.sharepoint.com: Specific vacation policy documentation including carry-over rules, PTO accrual rates, and request procedures

Both sources are marked as "Ready" and provide general availability to the Employee Self Service Agent for information retrieval.

Conversation Flow

The "Prompt Vacation Days" topic implements a sophisticated flow:

  1. Trigger Detection: Recognizes queries about vacation, PTO, time off, balance checks
  2. User Input Collection: Prompts for user email to personalize responses
  3. Generative AI Query: Uses "Create generative answers" node to search SharePoint knowledge base
  4. Data Processing: Extracts vacation balance and relevant policy information
  5. Response Delivery: Provides specific vacation day count plus detailed policy context with source citations

Lessons Learned

Building this Copilot Studio agent taught me several valuable lessons:

  • Knowledge Source Quality: The accuracy of generative AI responses depends heavily on well-organized, up-to-date source documents in SharePoint
  • Topic Design: Breaking conversations into focused topics (greeting, vacation queries, etc.) creates more maintainable and reliable agents
  • User Input Validation: Prompting for structured inputs like email addresses ensures the agent can retrieve personalized information
  • Context Preservation: Implementing "Start Over" functionality is crucial for users who want to ask multiple unrelated questions
  • Testing Importance: Thorough testing with diverse query phrasings reveals edge cases and improves trigger recognition
